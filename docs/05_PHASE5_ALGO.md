## Phase 5: 경로 탐색 알고리즘

### 5.1 가중치 샘플링
- [ ] **Dirichlet 분포 샘플링**
  - Dir(1, 1, 1)에서 20개 가중치 조합 생성
  - (α, β, γ) where α + β + γ = 1

### 5.2 A* 알고리즘 구현
- [ ] **휴리스틱 함수 설계**
  - 목표 곡선까지의 추정 거리
  
- [ ] **비용 함수 통합**
  - F(P) = α·f̃₁ + β·f̃₂ + γ·f̃₃
  
- [ ] **경로 탐색**
  - 각 (가중치, θ) 조합에 대해 최적 경로 탐색
  - 총 120개 후보 생성 (20 가중치 × 6 회전)

### 5.3 Pareto Set 생성

- [ ] **Non-dominated 경로 필터링**
