## Phase 3: 도형 처리

### 3.1 템플릿 관리
- [ ] **Polyline 템플릿 정의**
  - 기본 도형 템플릿 (하트, 별, 원 등)
  - 알파벳/숫자 템플릿
  - JSON 또는 SVG 형식으로 저장

### 3.2 사용자 입력 도형 처리
- [ ] **좌표점 → Polyline 변환**
  - 사용자가 그린 좌표점 집합을 polyline으로 변환
  - 노이즈 제거 (선택적 smoothing)
  - 점 개수 최적화 (너무 많은 점은 단순화)

### 3.3 도형 변환
- [ ] **스케일링**
  - 사용자 지정 bounding box에 맞게 크기 조정
  - 템플릿: 정규화 좌표 → 지리 좌표
  - 사용자 입력: 이미 지리 좌표이므로 스케일링 불필요
  
- [ ] **회전**
  - 6방향 (60° 간격) 회전 변환
  - θ ∈ {0°, 60°, 120°, 180°, 240°, 300°}

- [ ] **좌표 변환**
  - 템플릿 좌표 → 지리 좌표 변환
  - 사용자 입력은 이미 지리 좌표 (변환 불필요)
