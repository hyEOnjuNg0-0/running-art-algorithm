## 프로젝트 개요

사용자가 지도에서 원하는 범위를 지정하고, 그리고 싶은 모양/목표 거리/허용 신호등 개수를 입력하면, 최적의 러닝 코스를 Pareto set 기반으로 최대 5개까지 제안하는 시스템

---

## 현재 진행 상태

| Phase | 상태 | 설명 |
|-------|------|------|
| Phase 1 | ✅ 완료 | UI 프로토타입 (Streamlit) |
| Phase 2 | ⬜ 대기 | 데이터 레이어 (OSM) |
| Phase 3 | ⬜ 대기 | 도형 처리 |
| Phase 4 | ⬜ 대기 | 비용 함수 구현 |
| Phase 5 | ⬜ 대기 | 경로 탐색 알고리즘 |
| Phase 6 | ⬜ 대기 | 통합 및 최적화 |

**상태 범례**: ⬜ 대기 | 🔄 진행중 | ✅ 완료

**현재 작업 중**: Phase 2 대기

**최종 업데이트**: Phase 1 완료 (UI 프로토타입)

---

## 작업 순서

```
Phase 1 (UI) ──────────────────────────────────────────────►
              │
              ▼
         Phase 2 (데이터) ─────────────────────────────────►
                          │
                          ▼
                     Phase 3 (도형) ───────────────────────►
                                    │
                                    ▼
                               Phase 4 (비용함수) ─────────►
                                                 │
                                                 ▼
                                            Phase 5 (알고리즘)
                                                         │
                                                         ▼
                                                    Phase 6 (통합)
```

- Phase 1 완료 후 Mock 데이터로 UI 동작 확인
- Phase 2~5는 순차적으로 진행하며 각 단계별 테스트 수행
- Phase 6에서 전체 통합 및 최종 테스트

### 실행 방법
```bash
# 의존성 설치
pip install -r requirements.txt

# 애플리케이션 실행
streamlit run app.py
```

### 테스트 실행
```bash
pytest tests/ -v
```
