# 🏃‍♂️ Running Art (GPX Art) Route Generator

> 도로 그래프 위에서 원하는 형태를 그리는 최적의 러닝 코스를 자동으로 생성하는 시스템

## 📌 프로젝트 개요

사용자가 지도에서 원하는 범위를 지정하고, 그리고 싶은 모양/목표 거리/허용 신호등 개수를 입력하면, **최적의 러닝 아트 코스를 제안**하는 시스템입니다.

## 🎯 문제 정의 (Why this problem?)

### 배경

**러닝 아트(GPX Art)**는 달리며 특정 형태를 그리는 창의적인 활동으로, GPS 추적을 통해 지도 위에 그림, 글자, 상징적인 모양 등을 그릴 수 있습니다.

### 현실적인 제약사항

그러나 실제로 러닝 아트를 실행하려면 다음과 같은 문제들이 있습니다:

- **임의 그래프 구조**: 도로망은 규칙적이지 않은 복잡한 그래프 형태
- **보행 불가능한 구간**: 고속도로, 사유지 등 실제로 달릴 수 없는 경로 존재
- **현실 제약**: 신호등, 비포장도로, 경사도 등 러닝에 불편한 요소들
- **정확한 매칭의 어려움**: 목표 형태와 일치하는 경로는 거의 없음

### 프로젝트 목표

> 실제 도로 데이터와 제약 조건 내에서 **주어진 목표 형태를 가장 잘 재현**하면서 **현실적인 러닝 아트 경로**를 생성

---
## 🔑 핵심 기능

- 사용자가 지도에서 원하는 영역 선택
- 목표 형태(그림, 글자 등) 입력
- 희망 거리 및 허용 신호등 개수 설정
- Pareto 최적화를 통한 다양한 경로 제안 (최대 5개)
- 형태 유사도, 러닝 친화성을 균형있게 고려한 결과 제공
